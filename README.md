# Korea Biotech CEO Forum 2026 - 참석자 명단 페이지

## 프로젝트 개요
Korea Biotech CEO Forum 2026의 공식 참석자 명단을 확인할 수 있는 랜딩 페이지입니다. 국내 바이오 산업을 이끄는 33개 기업, 34명의 리더들이 참석하는 포럼의 명단을 우아하고 인터랙티브한 방식으로 제공합니다.

## 🎨 디자인 콘셉트
- **컬러 팔레트**: 공식 브랜드 컬러 시스템 적용
  - 프로그램 블루 (#4D8DF5) - 메인 악센트
  - 프로그램 그린 (#2C5F4F) - 다크 그린
  - 골드 (#D4AF37) - 프리미엄 악센트
  - 스카이 블루 (#93B5E1) - 밝은 블루
  - 라임 그린 (#85AC0D) - 생동감 있는 악센트
  - 베이지 기본 (#F8F6F1) - 따뜻한 배경
- **다채로운 카드 시스템**: 8가지 컬러로 순환하는 참석자 카드
- **왼쪽 보더 악센트**: 각 카드에 4px 컬러 보더로 시각적 구분
- **그라데이션 배경**: 부드러운 베이지-화이트-그린 그라데이션

## 주요 기능

### ✅ 현재 구현된 기능
1. **브랜드 아이덴티티 통합**
   - 공식 포럼 로고 적용 (제주도 일러스트 포함)
   - 로고 크기 확대 (70px / 모바일 50px)
   - 홈 아이콘(🏠) 네비게이션
   - 브랜드 컬러 팔레트 8색 활용

2. **다채로운 카드 디자인**
   - 8가지 컬러가 순환하는 참석자 카드
   - 왼쪽 4px 컬러 보더 악센트
   - 호버 시 상승 효과 및 그림자
   - 각 카드마다 다른 컬러 번호 뱃지
3. **반응형 디자인**
   - 모바일, 태블릿, 데스크톱 모든 화면에 최적화
   - Tailwind CSS를 활용한 현대적인 UI/UX
   - 모바일에서 "CEO Forum" 줄바꿈 방지 처리
   - 글래스모피즘 디자인 (frosted glass effect)

4. **인터랙티브 명단 공개**
   - "명단 상세 확인하기" 버튼 (이모티콘 제거)
   - 블루-그린 그라데이션 버튼 디자인
   - 부드러운 애니메이션 효과 (fadeInUp, 높이 전환)
   - 카드별 순차적 등장 효과 (staggered animation)

5. **참석 기업 카드 UI**
   - 총 33개 기업, 34명 참석자 정보 표시
   - 기업명 (한글/영문), 참석자명, 직함 포함
   - SK바이오팜: 이동훈 (대표/CEO), 최윤정 (본부장)
   - 유한양행: 최영기 (연구소장)
   - 호버 효과: 카드 상승, 컬러별 그림자 효과
   - 2단 그리드 레이아웃 (모바일에서 1단)

6. **통계 대시보드**
   - 참여 기업 수 (33개), 대표자 수 (34명), 포럼 년도 (2026)
   - 3단 그리드 카드 형태의 시각적 정보 제공
   - 각 통계별 다른 컬러 적용 (블루, 그린, 골드)
   - 미묘한 그림자 효과로 고급스러운 디자인

7. **시각 효과**
   - 베이지-화이트-그린 그라데이션 배경
   - 8가지 컬러가 순환하는 카드 시스템
   - 호버 시 카드별 컬러 보더 강조
   - 부드러운 트랜지션 효과

8. **접근성 및 UX**
   - Noto Sans KR 웹폰트 적용 (한글 최적화)
   - 실시간 상태 표시 (Confirmed Attendees 배지)
   - 블루 핑 애니메이션으로 시선 유도
   - 포럼 로고 및 브랜드 요소 통합
   - 홈 아이콘(🏠) 네비게이션
   - "Korea Biotech CEO Forum 2026" 타이틀
   - 모바일 줄바꿈 최적화

## 기술 스택
- **HTML5**: 시맨틱 마크업
- **Tailwind CSS**: 유틸리티 기반 스타일링
- **Vanilla JavaScript**: 인터랙티브 기능 구현 + 동적 컬러 렌더링
- **Google Fonts**: Noto Sans KR 웹폰트
- **External Assets**: 포럼 공식 로고 이미지

## 미구현 기능
현재 이 페이지는 정적 명단 표시 페이지로서 모든 필수 기능이 구현되어 있습니다.

### 향후 개선 가능한 기능 (우선순위순)
1. **검색 기능**
   - 기업명, 대표자명으로 실시간 필터링
   - 자동완성 검색 UI

2. **정렬 옵션**
   - 기업명 가나다순 / 영문순 / 대표자명순 정렬
   - 정렬 방식 선택 드롭다운

3. **상세 정보 모달**
   - 기업 클릭 시 상세 정보 팝업
   - 기업 소개, 웹사이트 링크 등 추가 정보

4. **인쇄/다운로드 기능**
   - PDF 다운로드 버튼
   - 인쇄 최적화 CSS

5. **다국어 지원**
   - 한국어/영어 전환 기능

6. **데이터 관리 시스템**
   - RESTful Table API 연동
   - 관리자 페이지에서 명단 수정 가능

## 로컬 개발
1. 이 프로젝트는 정적 HTML 페이지이므로 별도 빌드 과정이 필요하지 않습니다
2. `index.html` 파일을 웹 브라우저에서 직접 열거나 로컬 서버로 실행하세요
3. CDN을 통해 Tailwind CSS와 웹폰트를 로드하므로 인터넷 연결이 필요합니다

## 배포
**Publish 탭**을 통해 원클릭 배포가 가능합니다. 배포 후 제공되는 URL을 통해 웹사이트에 접근할 수 있습니다.
